# Baton command reference

Run `baton --help` for the current command list and
`baton COMMAND --help` for exact flags. The retained commands are `list`,
`show`, `next`, `enroll`, `update`, `unenroll`, `start`, `stop`, and `close`.

Global options are `--repo owner/name` and `--json`. Every mutation supports
`--dry-run`. The CLI has no config, setup, output-format, field-selection,
policy, branch, pull-request, CI, review, merge, or delivery commands.
